  <p>Once upon a time...</p>
  <img src=""/> <!-- Fill me in when you're ready -->

<template id="my-template">
  <img src=""/>

<script>
  var tmpl = document.querySelector("#my-template");
  tmpl.content.querySelector("img").src = "corgi.gif";
  document.body.appendChild(tmpl.content.cloneNode(true));
</script>

Shadow DOM

The Shadow DOM provides style and markup encapsulation.

Take some HTML and CSS nodes, bundle them together

Styles are scoped to the node

HTML cannot be traversed by external javascript

Same tech as browser makers

It is the same technology used by browser makers to implement tags like <video> and <textarea>.

Lifecycle Callbacks

ready()
When a new instance is created. Do setup work here

enteredView()
When an element is added to the page

leftView()
When an element is removed from the page

attributeChanged(attrName, oldVal, newVal)
When one of an element's attributes changes

Lifecycle Callbacks


<polymer-element name="fav-food">
  <template>
    <h2>My favorite food is <span id="food"></span></h2>
  </template>
  <script>
    Polymer('fav-food', {
      ready: function() {
        this.$.food.textContent = 'bacon';
      }
    });
  </script>
</polymer-element>

Attributes


<polymer-element name="fav-color" attributes="color">
  <template>
    <h2>My favorite color is: {{ color }}</h2>
  </template>
  <script>
    Polymer("fav-color", {
      color: "Orange"
    });
  </script>
</polymer-element>
          

<fav-color color="Purple"></fav-color>
